# 小学数学出卷工具

给孩子的暑假数学练习卷生成器，冀教版数学一年级+三年级。

## 快速开始

```
node engine/build.js        # 构建 app.html
双击 app.html               # 打开工具
```

或者直接打开已构建好的 `app.html`。

## 使用方法

1. 选择年级 → 知识点标签自动加载
2. 勾选要考的知识点（可多选）
3. 调滑块控制应用题模板比例（0%=纯静态题，100%=纯模板生成）
4. 点「生成试卷」→ 预览
5. 点「下载Word」→ 得到 .doc 文件（含答案页）

## 项目结构

```
math-exam-generator/
├── app.html                    ← 单文件应用（双击即用）
├── app/template.html           ← HTML 模板
│
├── bank/math/                  ← 题库（按年级分目录）
│   ├── grade1-shang/           ← 一年级上册
│   ├── grade1-xia/             ← 一年级下册
│   └── grade3/                 ← 三年级下册
│
├── engine/
│   ├── filler.js               ← 填槽引擎（模板→成品题）
│   ├── generator.js            ← 出卷引擎（选题→混排→输出试卷）
│   ├── build.js                ← 构建脚本（题库+模板→app.html）
│   ├── test-filler.js          ← 填槽测试
│   └── test-generator.js       ← 出卷测试
│
└── FORMAT.md                   ← JSON 格式规范
```

## 题库数据

| 年级 | 知识点 | 静态题 | 模板题 | 来源 |
|---|---|---|---|---|
| 一上 | 7 个 | 20 | 3 | 21世纪教育网、教习网 |
| 一下 | 7 个 | 20 | 6 | 21世纪教育网 |
| 三下 | 9 个 | 45 | 10 | 21世纪教育网、教习网 |

所有题目均来自网上真实教辅资源（21世纪教育网、教习网），经搜索引擎摘要提取。

## 技术栈

- 题库存储：JSON
- 填槽引擎：纯 JS（沙箱表达式求值）
- 出卷工具：单文件 HTML（内联所有 JS + 数据）
- 输出格式：Word（.doc，HTML格式，Word可直接打开）

## 教材版本

- 一年级：冀教版 2024 新版（上下册）
- 三年级：冀教版 2024 新版（下册，2025-2026 学年）
