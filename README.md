# 小学数学出卷工具

给孩子的暑假数学练习卷生成器，冀教版数学一年级+三年级。

## 设计思路

- **题目来源**：网上正规题源（梅学堂、51教学资源网、学科网等）的冀教版原题
- **AI 加工**：一次性格式化 + 打标签（难度/易错/题型/知识点）+ 适度修改
- **出卷方式**：本地网页工具，选条件 → 点生成 → 下载 Word，零 AI 消耗
- **省 Token**：题库建好后，每次出卷不调 AI，完全本地运行

## 技术栈

- 题库存储：JSON
- 出卷工具：单文件 HTML（双击即用）
- 输出格式：Word（.docx）

## 教材版本

- 一年级：冀教版 2024 新版
- 三年级：冀教版 2024 新版（2025-2026 学年）
